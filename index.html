Welcome!

I am an assistant professor in the Department of Computer Engineering at San José State University. I am interested in applying computational techniques to understand biological mechanisms. The tools I use to understand biological systems are data science tools such as machine learning and visualization.

Curriculum Vitae

Research Interests

  • Natural Language Processing: Language modeling, understanding, and generation using deep learning.
  • Bioinformatics Using computational tools to understand biological systems.
  • Computing Education: Using computational tools to improve learning.
  • Visualization: Exploring and creating visualizations to understand data.

Current Projects

  • Visualizing 3D Genomic Data
    Create visualizations using a generalizable framework.
  • Reasoning with Large Language Models
    Adding reasoning capabilities for complex tasks to large language models.
  • Improving Resolution of 3D Genomic Data
    We are developing a Machine Learning methods to improve the resolution of 3D genomic data.
  • Understanding Biological Mechanisms with 3D Genomic Data We are investigating how the heirarchical folding of the genome influences gene expression and contributes to diseases.
